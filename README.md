# Portfolio Website

Personal portfolio website showcasing my projects, skills, and experience as a Full-Stack Web Developer.

## Live Demo

https://yourdomain.com

## Tech Stack

- React 19
- Vite 6
- Plain CSS (custom design system with dark/light mode tokens)
- React Icons
- Web3Forms (contact form)

## Features

- Responsive design — mobile, tablet, and desktop
- Dark / light mode toggle
- Project showcase with GitHub links
- Working contact form with spam protection
- SEO optimization (meta tags, Open Graph)
- Smooth scroll navigation with active section tracking

## Installation

```bash
git clone https://github.com/Tilahun-git/portfolio.git
cd portfolio
npm install
npm run dev
```

Create a `.env` file in the root and add your Web3Forms key:

```env
VITE_WEB3FORMS_ACCESS_KEY=your_key_here
```

Get a free key at [web3forms.com](https://web3forms.com/).
